## v0.2（当前）

相对 v0.1-trial 的主要变化：

- 触发键改为右 Command（`triggerKey`，可配置），与豆包语音键（右 Ctrl）分离；工具不再监听用户的右 Ctrl。
- 任意键结束：语音激活期间按任意常规键即结束并切回鼠须管（豆包原生停语音）。
- 合成右 Ctrl 加 0.1s 停留（dwell），修复“语音唤起方式调整”弹窗。
- 加入 app 图标（mono-light 单色“豆”字标）。
- 清理死代码（右 Ctrl 防抖、停止/Esc 关弹窗链路、脚手架）与开发过程文档。

对应 tag：`v0.2.0`（已合并到 `main`）。

## 分支规则

`main` 保持为稳定主线。

后续实验功能不要直接在 `main` 上开发，应从 `main` 新建 `codex/` 前缀分支。

示例：

```bash
git switch -c codex/focus-bounce
```

如果实验失败，可以丢弃实验分支，保留 `main` 和 `v0.1-trial` 的稳定状态。
